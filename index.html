Signature-Based Intrusion Detection System

Detecting network threats with real-time signature matching and intelligent alerts.

Overview

This project implements a Signature-Based Intrusion Detection System using Python and packet analysis. It uses a database of attack signatures to detect malicious traffic in real-time.

Configuration

Instructions for setting up the environment, installing dependencies, and running the IDS tool.

  • Install Python 3.x
  • Install Scapy and Wireshark
  • Run the script with admin privileges
